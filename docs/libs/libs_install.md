#### Package Installer
```
npm install -g pnpm
```
#### Install Typescript

```
 pnpm install -D typescript
```

#### Install lib Enviroment Variables

```
pnpm i dotenv
```

#### Install Redis Client

```
pnpm i ioredis
```

#### Install dependences
```
pnpm i nodemon @aws-sdk/client-s3 drizzle-orm express mysql2
```

```
pnpm i --save-dev @types/node
```
